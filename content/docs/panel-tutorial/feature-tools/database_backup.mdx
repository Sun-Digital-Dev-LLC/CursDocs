## 如何備份
點開控制面板上的 BackUps 選項，點選下方資料庫備份的按鈕。

<img src="https://cdn-r2.curshosting.net/open_dbbackup.webp" />

點開後選擇需要備份的資料庫，按下建立後備份就建立好了。

<img src="https://cdn-r2.curshosting.net/selectdb.webp" />

如果資料有遺失或需要洗白時，就可從這裡下載下來導入進資料庫即可。

<img src="https://cdn-r2.curshosting.net/download_db.webp" />
