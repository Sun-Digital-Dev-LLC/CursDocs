## 如何翻譯
在控制面板上選擇一個文件，點開後先設定翻譯的語言。

<img src="https://cdn-r2.curshosting.net/select_file.webp" />
<img src="https://cdn-r2.curshosting.net/setting_tranlang.webp" />

設定完之後選取需要翻譯的文字，右鍵點選翻譯。

<img src="https://cdn-r2.curshosting.net/select_lang.webp" />
<img src="https://cdn-r2.curshosting.net/tranlation_content.webp" />

翻譯完成後會出現 **原始內容** 及 **翻譯後內容**，若翻譯無錯誤點選 **接受按鈕** 來完成翻譯內容。

<img src="https://cdn-r2.curshosting.net/accept_translation.webp" />
