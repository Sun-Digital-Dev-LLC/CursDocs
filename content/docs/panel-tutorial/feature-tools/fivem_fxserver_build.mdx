<Callout title="重要提醒" type="info">伺服器一定要先停止才可以進行選擇。</Callout>

## 如何部屬
選擇控制面板上 Settings 選項，往下滑會看到 FiveM 伺服器版本。

<img src="https://cdn-r2.curshosting.net/select_settings.webp" />

可以看到有版本、發布日期、支援到期日期、狀態。

<img src="https://cdn-r2.curshosting.net/select_fivem_version.webp" />

<Callout title="重要提醒" type="warn">選好版本後系統會自動幫您重新安裝新的版本，這個重新安裝並不會將您的檔案刪除。</Callout>
