當您在整理或修改檔案時不小心刪除到重要的核心檔案，以下將會教您如何復原：

## 如何復原？

拿 FiveM 伺服器作為舉例，FiveM 的核心資料夾為 `alpine`。

<img src="https://cdn.curshosting.net/ServerCore/sc-01.webp" />

若將核心資料夾刪除後，伺服器將無法成功開啟。

<img src="https://cdn.curshosting.net/ServerCore/sc-02.webp" />

需要復原核心資料夾須點選 **設定**，再點選 **重新安裝伺服器**。

<img src="https://cdn.curshosting.net/ServerCore/sc-03.webp" />

這裡按下確定重新安裝。

<img src="https://cdn.curshosting.net/ServerCore/sc-04.webp" />

開始重新安裝後會跳出等待畫面。
<Callout title="重要提醒" type="warn">這時候伺服器在面板上是無法進行任何操作的，請注意！！</Callout>

<img src="https://cdn.curshosting.net/ServerCore/sc-05.webp" />

重新安裝畫面。

<img src="https://cdn.curshosting.net/ServerCore/sc-06.webp" />

重新安裝完之後，會在控制台看到底下寫著 `install complete` 代表您的核心檔案回來了！

<img src="https://cdn.curshosting.net/ServerCore/sc-07.webp" />

以上是伺服器核心檔案復原的完整教學。如果在操作過程中有任何疑問，歡迎隨時至 Discord 聯絡我們！
