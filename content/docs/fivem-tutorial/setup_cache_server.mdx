</Callout>

## 設定快取伺服器
拿到設定的參數後請打開 `server.cfg` 輸入以下這兩行資料。

```cfg
fileserver_add ".*" "貼上拿到的資源網址"
adhesive_cdnKey "貼上拿到的 CDN KEY"
```

設定完成後就可開啟伺服器囉！
